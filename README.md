这是一个rpc的demo,实现技术为spring boot + netty +zookeeper。
分为服务服务提供者和服务消费者，流程：
1.生产者：
  1.1 扫描服务类注解，缓存服务
  1.2 启动服务通信器：netty服务端
  1.3 服务注册在zookeeper数据节点
  1.4 通过反射，本地调用服务方法
2.消费者:
  2.1 创建服务接口的代理
  2.2 在zookeeper中发现服务
  2.3 遍历生产者列表,建立netty客户端，远程调用生产者来完成任务

  这个demo使用了自定义注解，反射，动态代理等语言特性，spring boot，netty等框架。
